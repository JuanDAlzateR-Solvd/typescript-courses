# TypeScript Courses

Personal learning repository. Contains exercises and demos from TypeScript courses.

---

## Courses

| Folder | Topic |
|---|---|
| [`typescript-foundations/`](./typescript-foundations/) | TypeScript language fundamentals |
| [`playwright-for-typescript/`](./playwright-for-typescript/) | End-to-end testing with Playwright |

---

## Course Content

### TypeScript Foundations

| Module | Topic |
|---|---|
| 01 | Getting started with TypeScript |
| 02 | Core types |
| 03 | Union, literal, and enum types |
| 04 | Function patterns |
| 05 | The TypeScript compiler (`tsc`) |

Demos are `.ts` files located in each module's `demos/` folder.

### Playwright for TypeScript

| Module | Topic |
|---|---|
| 01 | Introducing Playwright |
| 02 | Core concepts and fixtures |
| 03 | Element targeting and assertions |
| 04 | Actions, navigation, and the Locator API |
| 05 | Configuring and running Playwright tests |
| 06 | Network interception and debugging tools |

Demos are `.spec.ts` files located in each module's `demos/` folder.

---

## Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [TypeScript](https://www.typescriptlang.org/) — `npm install -g typescript`
- [Playwright](https://playwright.dev/) — required only for the `playwright-for-typescript` course
